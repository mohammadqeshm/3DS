# 🌌 AtmosSim V2.4 Pro — شبیه‌ساز سه‌بعدی و واقع‌گرایانه اقلیم جوی ساحل خزر

یک محیط شبیه‌سازی بصری سه‌بعدی و بلادرنگ (Real-Time) فوق‌العاده با استفاده از موتور قدرتمند **Three.js** و **React** که به کاربران اجازه می‌دهد پارامترهای مختلف جوی، اقیانوسی و صوتی محیط ساحلی دریای خزر را دستکاری و تحلیل کنند.

---

## 🎨 ویژگی‌های برجسته (Key Features)

### ۱. سیستم ابری سه‌لایه پیشرفته (Dynamic Three-Layer Clouds Grid)
* **امواج سیروس (Cirrus - لایه بالا در ارتفاع ۱۱۰ متری):** ابرهای نازک و پر‌مانند مرتفع جوی.
* **توده‌های آلتوکومولوس (Altocumulus - لایه میانی در ارتفاع ۷۰ متری):** ساختار پفکی گسترده با ظاهری شگفت‌انگیز.
* **کومولوس تراکمی (Cumulus - لایه پایینی در ارتفاع ۳۵ متری):** ابرهای سه‌بعدی متراکم با سایه‌زنی نوری غلیظ و واقع‌گرایانه.

### ۲. فیزیک پویای جریان‌های باد (Real-Time Wind Vector Dynamics)
* تأثیر بردار وزش باد بر حرکت تصادفی تکان درختان و اهتزاز منظم بیجک پرچم ساحلی.
* اعوجاج فیزیکی امواج شبیه‌سازی‌شده در سطح آب دریا متناسب با سرعت باد.
* تأثیر سرعت باد بر فرکانس خروجی نویز سینتی‌سایزر صوتی.

### ۳. کنترل موقعیت خورشید و مه‌گرفتگی (Atmospheric & Solar Controls)
* **تنظیمات پیش‌فرض نوری جو:** طلوع خورشید (Sunrise)، ظهر (Noon)، غروب آفتاب (Sunset)، شفق گرگ‌ومیش (Twilight) و تاریکی شب (Night).
* **مه گرانشی چگال (Fog Density):** تجمع مه غلیظ در افق دریا و کاهش تدریجی میدان دید بر اساس ارتفاع.
* **ذرات آئروسل غبار (Dust & Haze):** شبیه‌سازی ذرات خاکی و شنی معلق در محدوده اتمسفر ساحل.

### ۴. دو حالت هوشمند ناوبری دوربین (Dual-Navigation Camera Matrix)
* **🚶 حالت اول شخص (Walkthrough):** قدم زدن بر روی ماسه‌های ساحلی با اعمال شتاب جاذبه واقعی زمین.
* **🛸 حالت پرواز آزاد (Free Fly):** امکان تغییر نامحدود زاویه دید و جابجایی بین اعماق بی‌پایان دریا یا پرواز تا بالاترین لایه‌های جو.

### ۵. سنتز زنده صوتی (Physical Audio Noise Synthesizer)
* استفاده از موتور موجی نویز فیلترشده مِنا با فیلتر فرکانسی کم‌گذر صوتی که تلاطم و شلاق باد را بر روی میکروفون‌های فرضی محیطی به‌طور زنده سنتز می‌کند.

## 🛠️ فناوری‌های مورد استفاده (Technology Stack)

* **Engine:** Three.js / @types/three (WebGL Core)
* **UI/UX:** React 19 + Tailwind CSS (Vite Engine)
* **Animations:** Motion (React)
* **Bundler & Tooling:** Vite, ESBuild, TypeScript

## 🚀 راهنمای راه‌اندازی و اجرا (Installation & Run)

ابتدا مطمئن شوید که نسخه پایدار **Node.js** بر روی سیستم نصب شده باشد. سپس مراحل زیر را دنبال کنید:

### ۱. نصب وابستگی‌های پروژه
```bash
npm install
```

### ۲. اجرای نسخه توسعه محلی (Development Server)
برنامه را با آدرس پیش‌فرض پورت ۳۰۰۰ به صورت محلی بالا بیاورید:
```bash
npm run dev
```
سپس مرورگر خود را باز کرده و وارد آدرس زیر شوید:
`http://localhost:3000`

### ۳. کامپایل و ساخت نسخه نهایی (Production Build)
جهت بهینه‌سازی کدهای کلاینت برای استقرار بر روی سرورهای ابری:
```bash
npm run build
```

## 🎮 راهنمای کلیدها و میانبرهای حرکتی (Controls Matrix)

* **حرکت در محیط:** `W, A, S, D` یا کلیدهای مکان‌نما (Arrow Keys)
* **صعود / سقوط (مخصوص حالت پرواز آزاد):** `Space` (بالا) / `Shift` (پایین)
* **چرخش دوربین سه‌بعدی:** کلیک چپ ماوس و حرکت دادن آن (Drag)
* **کنترل‌های جوی:** استفاده از پنل شیشه‌ای تعاملی منوی پیکربندی برای تغییر توده‌های ابری، جهت وزش و میزان صدا.
