# 🛸 AtmosSim V2.4 Pro — Windows x84_64 Electron Compilation Manual
## راهنمای ساخت نرم‌افزار تحت ویندوز ۱۰ نسخه ۶۴ بیتی با استفاده از الکترون

این پوشه حاوی تمام پیکربندی‌های لازم برای تبدیل شبیه‌ساز سه‌بعدی اقلیم **AtmosSim V2.4 Pro** به یک برنامه بومی (Native Desktop Application) برای ویندوز ۶۴ بیتی است. با دنبال کردن این راهنما به صورت خودکار قادر خواهید بود پروژه را روی سیستم خود به فایل نصبی `.exe` یا نسخه پرتابل فشرده متصل کنید.

---

## 📋 پیش‌نیازها (Prerequisites)

قبل از اقدام به ساخت، مطمئن شوید نرم‌افزارهای زیر روی سیستم عامل ویندوز شما نصب شده‌اند:
1. **Node.js (نسخه ۱۸ یا بالاتر):** از سایت رسمی [Node.js](https://nodejs.org) دانلود و نصب کنید.
2. **Git (اختیاری):** جهت کپی کردن کدهای پروژه.

---

## 🛠️ مراحل ساخت برنامه روی سیستم شما (Build Steps)

### مرحله ۱: دانلود فایل‌های پروژه
کدهای پروژه را دانلود کرده و یک ترمینال (Command Prompt یا PowerShell) در پوشه اصلی پروژه باز کنید.

### مرحله ۲: نصب ملزومات و پکیج‌ها
دستور زیر را در ترمینال اجرا کنید تا تمام پیش‌نیازها (از جمله React, Three.js, Electron و Core Builder) نصب شوند:
```bash
npm install
```

### مرحله ۳: اجرای آزمایشی برنامه روی دسکتاپ (Development Mode)
قبل از خروجی نهایی، می‌توانید برنامه را به صورت زنده در پنجره دسکتاپ الکترون بررسی کنید:
```bash
npm run win:dev
```
*این دستور پروژه را به همراه اتمسفر و فیزیک باد در یک پنجره مستقل بومی ویندوز باز می‌کند.*

### مرحله ۴: ساخت خروجی نهایی ویندوز ۶۴ بیتی (Production Package)
برای بسته‌بندی پروژه و ایجاد فایل نصب نهایی ویندوز، تک دستور زیر را اجرا کنید:
```bash
npm run win:build
```

## 📦 خروجی‌ها کجا ذخیره می‌شوند؟ (Output Artifacts)

پس از اتمام موفقیت‌آمیز عملیات ساخت، یک پوشه جدید به نام `dist_windows` در ریشه پروژه ایجاد می‌شود که حاوی فایل‌های زیر است:

1. **`AtmosSim V2.4 Pro Setup 0.0.0.exe`**: فایل نصبی آسان و هوشمند ویندوز (NSIS Installer) با قابلیت ایجاد شورتکات روی دسکتاپ و منوی استارت.
2. **`AtmosSim V2.4 Pro-0.0.0-win-x64.zip`**: نسخه فشرده بدون نیاز به نصب (Portable) مخصوص سیستم‌های ۶۴ بیتی ویندوز ۱۰ و ۱۱.

## 🎨 ساختار پیکربندی الکترون (Electron Configurations)

* **`win64/main.js`**: کنترل‌کننده فرآیند اصلی دسکتاپ، تنظیم ابعاد پنجره (1280x720)، چسباندن لوگوی نرم‌افزار و پنهان کردن هدر مشکی پیش‌فرض ویندوز جهت محیط کاربری غوطه‌ورکننده (Immersive View).
* **`win64/preload.js`**: پل امنیتی میان محیط محلی سیستم‌عامل و هسته مرورگر.
* **`win64/electron-builder.json`**: پیکربندی موتور بسته‌بندی نهایی که به طور استاتیک خروجی `dist` را فشرده می‌کند و آیکون باکیفیت طراحی‌شده اقلیم را به فایل نصبی الصاق می‌نماید.
* **`src/assets/images/app_icon_1781486855743.jpg`**: آیکون مدرن و رندر شبیه‌ساز پیشرفته باد و ابر که به عنوان لوگوی رسمی ویندوز استفاده می‌شود.
